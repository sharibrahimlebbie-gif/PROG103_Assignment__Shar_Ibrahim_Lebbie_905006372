# Student Record Management System (SRMS)

## Overview
The Student Record Management System (SRMS) is a terminal-based Python application developed for the PROG103 Principles of Structured Programming assignment.

The system is designed to help manage and process student academic records in a simple, organized, and efficient manner.

---

## Features
- Add student records
- View all student records
- Search for student records
- Update student information
- Delete student records
- Generate performance summary
- Input validation for accurate data entry
- Menu-driven terminal interface

---

## Programming Concepts Used
This project demonstrates the use of:
- Functions
- Variables and data types
- Lists and dictionaries
- Conditional statements
- Loops
- Input validation
- Structured programming principles
